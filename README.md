# Web API .NET 9 – CRUD de Produtos

Este projeto é uma API REST desenvolvida com .NET 9 utilizando Entity Framework Core e SQL Server.

## Tecnologias utilizadas

- .NET 9
- Entity Framework Core
- SQL Server
- REST API

## Funcionalidades

- Listar produtos
- Buscar produto por ID
- Criar produto
- Atualizar produto
- Deletar produto
